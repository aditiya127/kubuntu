Desktop Environment digunakan untuk merujuk pada keseluruhan lingkungan kerja visual yang diakses oleh pengguna.

Discover

Discover adalah aplikasi manajer perangkat lunak di Kubuntu (dan lingkungan KDE lainnya) yang memudahkan pengguna untuk mencari, menginstal, dan mengelola perangkat lunak. Berikut adalah beberapa fitur utama Discover:

1. Memudahkan pengguna untuk menavigasi dan menemukan aplikasi.

2. Menyediakan informasi tentang aplikasi, termasuk deskripsi, tangkapan layar, dan ulasan dari pengguna lain.

Konsole

Konsole adalah terminal emulator di lingkungan desktop KDE, seperti Kubuntu. Ia memungkinkan pengguna untuk berinteraksi dengan sistem melalui baris perintah. Fitur utamanya meliputi dukungan multitab, kustomisasi tampilan, riwayat perintah, dan dukungan untuk berbagai shell. Konsole sangat berguna bagi pengguna yang memerlukan akses langsung ke sistem operasi untuk pengembangan, administrasi, atau tugas sehari-hari.

Kate

Kate adalah editor teks canggih di lingkungan KDE yang dirancang untuk pengeditan kode dan teks. Fitur utamanya meliputi sorotan sintaksis untuk berbagai bahasa pemrograman, dukungan multitab, dan kemampuan kustomisasi antarmuka. Kate juga mendukung plugin untuk menambah fungsionalitas, serta menyediakan fitur pencarian dan penggantian yang kuat. Aplikasi ini sangat cocok untuk programmer dan pengguna yang memerlukan alat pengeditan teks yang fleksibel dan fungsional.

1. Pengeditan Teks Umum: Kate dapat digunakan untuk membuat dan mengedit file teks biasa, seperti dokumen, catatan, atau skrip.

Spectacle

Spectacle adalah aplikasi penangkap layar (screenshot) di lingkungan desktop KDE. Berikut adalah beberapa fitur utama Spectacle:

1. Pengambilan Gambar: Memungkinkan pengguna untuk mengambil tangkapan layar dari seluruh layar, jendela tertentu, atau area yang dipilih.

2. Pengeditan Dasar: Setelah mengambil tangkapan layar, pengguna dapat melakukan beberapa pengeditan dasar, seperti menambahkan anotasi atau menggambar pada gambar.

3. Format Penyimpanan: Mendukung berbagai format penyimpanan gambar, termasuk PNG, JPG, dan lainnya.

4. Kustomisasi: Pengguna dapat mengatur pintasan keyboard untuk mengambil tangkapan layar dengan cepat.

5. Ekspor dan Berbagi: Memungkinkan pengguna untuk mengekspor hasil tangkapan layar ke lokasi tertentu atau membagikannya melalui clipboard. Spectacle adalah alat yang berguna bagi pengguna yang sering memerlukan tangkapan layar untuk dokumentasi, presentasi, atau berbagi informasi.

Info Center

Info Center adalah aplikasi di lingkungan desktop KDE yang memberikan informasi tentang sistem. Berikut adalah beberapa fitur utama Info Center:

1. Detail Sistem: Menampilkan informasi mendetail tentang perangkat keras dan perangkat lunak, seperti CPU, RAM, penyimpanan, dan perangkat jaringan.

2. Status Jaringan: Menyediakan informasi tentang koneksi jaringan yang aktif, termasuk alamat IP dan status koneksi.

3. Informasi Perangkat Keras: Memberikan data tentang komponen perangkat keras, seperti motherboard, kartu grafis, dan perangkat input.

4. Versi Perangkat Lunak: Menampilkan informasi tentang versi sistem operasi dan lingkungan desktop yang sedang digunakan.

5. Antarmuka Sederhana: Memiliki antarmuka yang mudah dinavigasi, memungkinkan pengguna untuk menemukan informasi yang mereka butuhkan dengan cepat.

Info Center adalah alat yang berguna bagi pengguna dan administrator sistem untuk memantau dan memahami konfigurasi dan status sistem mereka.

Gwenview

Gwenview adalah penampil gambar di lingkungan desktop KDE, seperti Kubuntu. Ia memungkinkan pengguna untuk membuka dan melihat berbagai format gambar, menavigasi antar gambar dengan mudah, dan melakukan pengeditan dasar seperti pemotongan dan pengubahan ukuran. Gwenview juga mendukung tampilan slide show dan menampilkan metadata gambar. Aplikasi ini dirancang untuk memberikan pengalaman yang sederhana dan efisien dalam mengelola koleksi gambar.

FireFox

Firefox adalah peramban web yang memungkinkan pengguna untuk menjelajahi internet. Fungsinya meliputi:

1. Penjelajahan Web: Memungkinkan akses ke situs web dan konten online.

2. Privasi dan Keamanan: Menyediakan fitur untuk melindungi privasi pengguna, seperti pelindung pelacakan dan mode penjelajahan pribadi.

3. Kustomisasi: Memungkinkan penyesuaian melalui tema dan ekstensi untuk meningkatkan pengalaman pengguna.

4. Sinkronisasi: Mengizinkan sinkronisasi bookmark, riwayat, dan pengaturan di berbagai perangkat.

Firefox dirancang untuk memberikan pengalaman menjelajah yang cepat, aman, dan dapat disesuaikan.

Dolphin

Dolphin adalah pengelola file default di lingkungan desktop KDE. Ia menawarkan antarmuka intuitif dengan dukungan multitab, memungkinkan pengguna untuk menjelajahi dan mengelola file dengan mudah. Fitur unggulannya mencakup pratinjau file, pencarian cepat, dan integrasi dengan aplikasi lain. Dolphin dirancang untuk memberikan pengalaman pengelolaan file yang efisien dan fleksibel bagi pengguna Linux.

Start

Start Menu berisi beberapa elemen:

Kolom aplikasi: Menampilkan aplikasi yang telah terinstal di komputer.

Pencarian: Kolom pencarian untuk mencari aplikasi, file, atau pengaturan.

Pengaturan cepat: Tautan ke pengaturan seperti Power (untuk mematikan atau restart), Settings, dan File Explorer.
